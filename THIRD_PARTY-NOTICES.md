# Third-party notices

Context Normalizer core has no third-party runtime dependency.

The Codex integration builds a pinned revision of [OpenAI Codex](https://github.com/openai/codex), which is distributed under the Apache License 2.0. The Context Normalizer release artifact contains integration files and a reviewed source patch. The installer obtains the pinned upstream source directly from its upstream repository.

The Pi integration has no runtime package dependency. Pi itself remains an independently installed client.
